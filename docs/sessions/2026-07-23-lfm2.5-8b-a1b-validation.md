## Findings (measured)

| Trial | Server expert placement | Bench tg | Peak VRAM | Claw-quick | Agentic wall | Status |
|---|---|---|---|---|---|---|
| A | `.*exps.*=CPU` (auto when `N_CPU_MOE=None`) | 174.0 t/s | 2.3 GB | 0.20 (1/5) | ~106 s | KEEP |
| B | `--n-cpu-moe 0` | 174.1 t/s | 6.5 GB | 0.20 (1/5) | ~38 s | KEEP |

- Only **T010_contact_lookup** passed in both runs (T002/T004/T006/T008 = 0 tool calls or fail).
- **Bench TPS did not move** A→B: Combined TPS is from `llama-cli`; trial A bench had no expert CPU override. Full-VRAM gain shows up as **lower agentic wall time** and **higher peak VRAM**, not tg.
- Prefight est ~6650 MB / limit 7900 OK both trials.

### Harness lesson
For MoE name markers (`A1B`, …): `N_CPU_MOE=None` ≠ “no offload”. It triggers full expert CPU dump on **server**. Use `N_CPU_MOE=0` when the GGUF fits physical VRAM.
