Inferência — cola rápida

Uma página. Cola na parede mental.

Definição

Inferência = gerar saídas com pesos já treinados (congelados). Não atualiza o modelo.

Treino vs inferência

TREINO      → muda pesos (fora deste curso)
INFERÊNCIA  → só calcula o próximo token (este curso)

Loop

prompt → tokens → modelo → próximo token → … → texto

Métrica que vais ver

t/s  ou  TPS  = tokens por segundo (velocidade de geração)

Neste repo

GGUF no disco + llama.cpp  =  inferência local
hf download                 =  obter pesos (ainda não é treino)
